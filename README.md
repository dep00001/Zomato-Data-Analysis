# Zomato Data Analysis

This project analyzes Zomato restaurant data using Python.

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Analysis Performed
Data cleaning
Exploratory Data Analysis
Restaurant ratings analysis
City wise restaurant distribution
Cuisine analysis

## Key Insights
Most restaurants have ratings between 3.5 and 4.5.
Some cities dominate the restaurant market.
Pricing has moderate relation with ratings.
